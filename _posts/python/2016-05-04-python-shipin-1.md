# 第一章

## python 基础知识

```
小数=浮点数
表达式：由运算符和操作数按照语法形式组合成的字符序列
+ 加
- 减
* 乘
/ 除
// 浮点除法，对结果四舍五入
% 求余（求模）
** 求幂
小数和整数运算=小数
小数和小数运算=小数
整数和整数运算=整数
1/2=0（正常等于0.5但是整数规则所以等于0）
1.0/2=0.5
L表述长整形（数字达到字符长度-2147483648到2147483647之外需要使用长整形）
整形和长整形运算等于长整形
python2.2版本以前整数达到长整形范围时候会报错
0x 表示16进行（0xFF为255，255+0xFF=510，）
0开头 表示8进制（010=8）
混合运算：
55+0xFF+012=300
```
