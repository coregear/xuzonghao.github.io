### 服务器端吐json时的标准header  
```
header('Content-type: application/json');
```

```
$tmp = json_encode(lookup_ip_source("61.135.153.140", false, 'UTF-8'));
var_dump($tmp."\n");
echo "<br>"; 
$foo = (json_decode($tmp, true));       //解析json                                                                                                                               
echo $foo['city'];
```
